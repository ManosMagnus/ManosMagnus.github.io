1.1 /brute-force.php

1.1.1 Summary

  • Host: dvws.local:8080
  • GET http://dvws.local:8080/authenticate-user-prepared HTTP/1.1
  • OPCODE: 1-TEXT
  • Outgoing Messages format: JSON strings
  • Incoming Messages format: HTML
  • Input encryption: BASE64
  • PING/PONG: None
  • Close message: OPCODE= CLOSE, TEXT= 1001
